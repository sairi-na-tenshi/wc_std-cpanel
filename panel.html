iPanel

Панелька с закруглёнными краями, тенью и двумя фоновыми цветами без использования картинок и скриптов (разве что только для инициализации).

Тут сайдбар скрыт или удалён

Для закругления используется техника описанная в статье "Кроссбраузерные закруглённые уголки", но лишняя VML вёрстка подключается с помощью скрипта, генерируемого посредством vml-suki путём слияния vml-файлов всех включённых в пакет модулей. О модуляризации можно почитать в статье "Организация пакетов с помощью css-suki". Аналогичный сборщик для скриптов - js-suki.

Осторожно, сайдбар подстраивается под размер содержимого И в сайдбаре и в основном блоке имеется отступ равный радиусу скругления

Основная сложность - двуцветный фон. Его было бы элементарно реализовать, раскрасив две панельки в разные цвета, если бы не скруглённые края, которые не умеют обрезать вложенные элементы. Поэтому сначала вся панель заливается градиентом от одного цвета к другому, а поверх рисуются две панели смещённые так, чтобы не залезать на скругления. Если приглядеться, то при маленькой ширине по краям можно заметить небольшие градиентные артефакты.

Для сайдбара есть специальная панель с табами Как правило контенту имеет смысл добавить динамический отступ от краёв с помощью iSpacer

Сильная совместимость: IE8+, Chrome 5+, FF3.5+, Opera 10.5+; слабая: IE6+

В IE всё делается с помощью VML. В Опере для градиентного фона используется SVG.
